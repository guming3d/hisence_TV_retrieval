Hisense TV sports AI assistant. Users press the remote AI key and ask sports questions in natural-language Chinese. The agent answers via seven tools: Foundry IQ program-library retrieval, Web IQ web grounding, EPG schedule lookup, live-score lookup, tune-to-channel, and cross-session viewer memory (remember/recall preferences). Scope: EPG schedules, live scores, player/team/competition knowledge, and sports-content recommendations.
